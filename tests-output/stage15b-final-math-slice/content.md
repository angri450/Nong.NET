# 1  Introduction

This test document was generated by the DocxCore library to verify that the ParagraphBuilder, TableBuilder, DocumentWriter, SectionBuilder, and StyleBuilder classes work correctly together. The library provides a fluent API for creating Word documents programmatically from .NET code.

# 2  Features Overview

## 2.1  ParagraphBuilder

The ParagraphBuilder class provides a fluent interface for creating styled paragraphs. It supports text, bold text, superscript text, alignment, indentation, line spacing, and page break controls. Paragraphs can be composed by chaining method calls in a builder pattern.

Note: This paragraph was built directly with ParagraphBuilder and appended to the body, bypassing DocumentWriter.

## 2.2  TableBuilder

The TableBuilder class creates tables with customizable columns, header rows, data rows, border styles (including three-line academic table format), and Word built-in table styles. It uses a fluent API similar to ParagraphBuilder.

## 2.3  DocumentWriter

The DocumentWriter class provides a high-level API for building complete academic documents. It supports automatic heading numbering, reference detection in body text (e.g., [1]), footnotes, endnotes, cross-references, tables of contents, and more.
