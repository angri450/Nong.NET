# Nong.NET Documentation

This directory holds repository-level maintenance documents that do not need to sit in the repository root.

## Current Documents

- `CAPABILITY.md` - current Nong CLI capability table and command inventory.
- `DEPENDENCY_CONTROL.md` - fork, source-snapshot, and dependency-control notes.
- `release-checklist.md` - release checklist for the current release line.
- `feedback/` - user feedback and repair notes grouped by date/topic.

Keep root-level files limited to project entry points and build metadata: `README.md`, `README.zh-CN.md`, `CLAUDE.md`, `AGENTS.md`, `LICENSE`, `.gitignore`, `Directory.Build.props`, and source project folders.
