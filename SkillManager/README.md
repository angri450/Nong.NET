# Angri450.Nong.Skill.Manager

Skill management CLI — validate, scan, package, eval, scaffold.

## Install

```bash
dotnet tool install -g Angri450.Nong.Skill.Manager
```

## Commands

| Command | Description |
|---------|-------------|
| `skill-manager validate <dir>` | Validate SKILL.md format |
| `skill-manager scan <dir>` | Security scan |
| `skill-manager package <dir>` | Package as .zip |
| `skill-manager eval <dir>` | Blind evaluation |
| `skill-manager scaffold <name>` | Scaffold new skill |
| `skill-manager inventory <dir>` | List files |
