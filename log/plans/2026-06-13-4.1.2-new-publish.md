# 4.1.2 new publish

Date: 2026-06-13
Status: in progress

## Context

Root `nupkg/` was cleaned and repacked for 4.1.0, but NuGet already contains all 7 current 4.1.0 packages. NuGet also has 4.1.1 for `Angri450.Nong.Cli` and `Angri450.Nong.Tool.Pdf`, so a publishable replacement must use a version higher than both lines.

Use `4.1.2` as the new release version.

## Scope

- Bump first-party project/package versions from 4.1.0 to 4.1.2.
- Bump `Cli/Common/CliVersion.cs` to 4.1.2.
- Update tests and current release docs that assert the CLI/package version.
- Keep `Cli/Common/OcrRuntimeVersion.cs` unchanged.
- Clean root `nupkg/`, repack only the 7 current tool packages, and run pack audit.
- Install all 7 packages from the freshly packed local source and run installed smoke checks.
- Publish the 7 4.1.2 packages to NuGet with `--skip-duplicate`.
- Record the result in changelog and commit.

## Release Package Set

- `Cli/NongCli.csproj` -> `Angri450.Nong.Cli`
- `Chart/tools/nong-chart.csproj` -> `Angri450.Nong.Tool.Chart`
- `Diagram/tools/nong-diagram.csproj` -> `Angri450.Nong.Tool.Diagram`
- `Pdf/tools/nong-pdf.csproj` -> `Angri450.Nong.Tool.Pdf`
- `Pptx/tools/nong-pptx.csproj` -> `Angri450.Nong.Tool.Pptx`
- `MultiModal/tools/nong-ocr.csproj` -> `Angri450.Nong.Tool.Ocr`
- `Imaging/tools/nong-imaging.csproj` -> `Angri450.Nong.Tool.Imaging`
