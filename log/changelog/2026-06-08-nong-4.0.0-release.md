## What Changed

- Unified all main Nong package versions on `4.0.0`.
- Updated CLI runtime version reporting to `4.0.0`.
- Updated literature provider User-Agent strings to `Nong-Literature/4.0.0`.
- Updated OCR runtime package tests and install-model test discovery to look for `4.0.0` runtime bundles.
- Updated README and README.zh-CN with the 4.0.0 release line.
- Updated `CAPABILITY.md`, `Cli/AGENT.md`, and `release-checklist.md` for the current 4.0.0 contract.
- Exposed the implemented `word table-reflow` command in `commands --json`, bringing the manifest to 93 implemented commands.

## Contract Notes

- The shared slice contract remains `nong-pandoc/package/v1`.
- `word format-audit` is the visible formatting quality gate for Word documents, including CI flags `--fail-on-warning` and `--min-score`.
- OCR runtime mainline remains Windows x64 for this release unless the maintainer explicitly overrides the runtime policy.
